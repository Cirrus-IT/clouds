A column element to be used inside the triple column block. It contains an SVG icon, a title and a bulleted list.

  • The element heading is a h2 element with a class of text__title.
  • The bulleted list is a simple unordered list (ul).
  • Text inside a li element sits inside a p tag.

A list of available SVG icons is listed in the Typography section.
