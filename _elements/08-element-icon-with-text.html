A column element to be used inside the triple column block. It contains an SVG icon, a title and a bulleted list with checks.

  • The element heading is a h2 element with a class of text__title.
  • The text are simple p tags.
  • Optionally you can add a "Find out more" link below the list

A list of available SVG icons is listed in the Typography section.
