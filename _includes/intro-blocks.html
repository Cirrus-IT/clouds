Blocks

Each block takes up the full width of the viewport. A block can contain several sections, depending on the amount of columns/elements a block has been allocated.

All blocks take a minimum of two classes: block and block--[BLOCK-NAME].

The following block names are available to use:

  • block--hero
  • block--single
  • block--double
  • block--triple
  • block--testimonial
  • block--blog

Optionally a block can have a third class indicating the background shading of the section. The background colour will stretch the full width of the viewport.

  • light
  • dark
