There are three available headings:

  • h1
  • h2
  • h3

The h2 heading can have an additional class of text__title.

show_code: true ---

h1: Main heading. Use in a hero block

h2: Secondary heading. Use in a blog post

h2: Section title. Has a special class, use as an element heading

h3: Small heading. Use as a sub title in a text element.
